# salary_predictor_

Linear Regression model to predict salary from years of experience.

Built using :
Python, Pandas, NumPy, Scikit-learn, Matplotlib.

## What makes this different from basic regression:
- Used real messy data with missing values (NaN)
- Handled missing values using median imputation
- Applied 80/20 train-test split for honest accuracy evaluation
- Tested on 4 completely unseen data points

## Results:
- R² score: 98.7% on unseen test data
- Predictions within ~3,000 of actual salaries

## ML Concepts covered:
- Linear Regression
- Data Cleaning (median imputation)
- Train-Test Split
- Model Evaluation on unseen data
- Data Visualization
